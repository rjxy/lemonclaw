## ✨ Features

- **资讯监控** — 通过 RSS 自动抓取前端技术、体育赛事、游戏动态等多领域热点
- **AI 总结** — 长文本压缩 + AI 提炼核心要点，生成简洁摘要
- **飞书联动** — 自动推送至飞书群 & 归档至飞书文档
- **知识库** — 向量化存储，构建可检索的知识储备
- **智能问答** — 飞书群 @机器人 即可基于历史资讯精准回复
- **自然语言定时任务** — 飞书内用自然语言创建个性化定时任务
